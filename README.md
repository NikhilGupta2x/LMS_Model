# Edemy — Learning Management System 🎓

A full-stack LMS built with the MERN stack where students can purchase & learn from courses and educators can create & manage content.

🔗 **Live Demo:** [lms-model-pink.vercel.app](https://lms-model-pink.vercel.app)

---

## Tech Stack
- **Frontend:** React.js, Tailwind CSS, Clerk (auth)
- **Backend:** Node.js, Express.js, MongoDB
- **Services:** Stripe (payments), Cloudinary (media), Clerk (webhooks)
- **Deployment:** Vercel (client) + Render (server)

---

## Features
- Student: browse, purchase, and access courses
- Educator: create and manage courses & lectures
- Secure payments via Stripe
- Media uploads via Cloudinary

---

## Local Setup

```bash
cd client
npm install
npm run dev
```
